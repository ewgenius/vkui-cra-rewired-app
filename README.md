# VKUI Test App

Uses [react-app-rewired](https://github.com/timarney/react-app-rewired)
Resolves `@vkontakte/vkui` to dev bundle in `../VKUI/dist` relative to app root

- place next to VKUI folder
- inside VKUI:
  - run `yarn dev`
  - run `yarn tsc` to prepare typings
- inside test app:
  - run `yarn start`
